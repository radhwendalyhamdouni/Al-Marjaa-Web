# 🌙 خادم المرجع - Al-Marjaa Web

أول إطار عمل ويب عربي متكامل وسهل الاستخدام.

## ✨ المميزات

- 🚀 **خادم HTTP/HTTPS سريع** - مبني على Hyper
- 📡 **WebSocket** - دعم الاتصال في الوقت الحقيقي
- 🛤️ **نظام توجيه عربي** - سهل وبديهي
- 🔐 **جلسات وملفات تعريف** - آمنة ومتقدمة
- 📝 **قوالب HTML** - بناء صفحات ديناميكية
- 📦 **JSON تلقائي** - تحويل تلقائي للبيانات
- 🛡️ **حماية متقدمة** - CORS, Rate Limit, CSRF
- 🌍 **عربي بالكامل** - واجهات برمجة عربية

## 📦 التثبيت

أضف إلى `Cargo.toml`:

```toml
[dependencies]
almarjaa-web = { git = "https://github.com/radhwendalyhamdouni/Al-Marjaa-Web", branch = "main" }
```

## 🚀 البدء السريع

### خادم بسيط

```rust
use almarjaa_web::خادم;
use almarjaa_web::json;

#[tokio::main]
async fn main() {
    خادم::جديد()
        // صفحة رئيسية
        .مسار("/", |طلب| {
            طلب.رد_بhtml("
                <h1>مرحباً بالعالم!</h1>
                <p>هذا خادم المرجع</p>
            ")
        })
        // API بسيط
        .مسار("api/معلومات", |طلب| {
            طلب.رد_بjson(json!({
                "الاسم": "خادم المرجع",
                "الإصدار": "3.4.0",
                "اللغة": "عربي"
            }))
        })
        // مسار مع معامل
        .مسار("مستخدم/{الاسم}", |طلب| {
            let name = طلب.معامل_أو("الاسم", "زائر");
            طلب.رد_بنص(format!("أهلاً بك، {}!", name))
        })
        // تشغيل الخادم
        .شغل(8080)
        .await;
}
```

### طرق HTTP مختلفة

```rust
خادم::جديد()
    // GET
    .جلب("مقالات", |طلب| {
        طلب.رد_بjson(json!({"مقالات": []}))
    })
    // POST
    .إرسال("مقالات", |طلب| {
        let body = طلب.المحتوى_كjson_خام().unwrap();
        طلب.رد_بjson(json!({"تم": true, "البيانات": body}))
    })
    // PUT
    .تحديث("مقالات/{معرف}", |طلب| {
        let id = طلب.معامل("معرف").unwrap();
        طلب.رد_بنص(format!("تم تحديث المقالة {}", id))
    })
    // DELETE
    .حذف("مقالات/{معرف}", |طلب| {
        let id = طلب.معامل("معرف").unwrap();
        طلب.رد_بنص(format!("تم حذف المقالة {}", id))
    })
    .شغل(8080).await;
```

### الوسائط (Middleware)

```rust
use almarjaa_web::وسیطة;

خادم::جديد()
    // تسجيل الطلبات
    .وسیطة(|طلب| {
        println!("[{}] {} {}", 
            chrono::Local::now().format("%H:%M:%S"),
            طلب.الطريقة,
            طلب.المسار
        );
        None // استمرار المعالجة
    })
    // التحقق من API Key
    .وسیطة(|طلب| {
        if طلب.رأس("X-API-Key").is_none() {
            return Some(رد::غير_مصرح("مفتاح API مطلوب"));
        }
        None
    })
    .مسار("api/بيانات", |طلب| {
        طلب.رد_بjson(json!({"بيانات": "سرية"}))
    })
    .شغل(8080).await;
```

### CORS

```rust
خادم::جديد()

```

### معاملات الاستعلام (Query Parameters)

```rust
خادم::جديد()
    .مسار("بحث", |طلب| {
        let q = طلب.استعلام_أو("q", "");
        let page = طلب.استعلام_أو("page", "1");
        
        طلب.رد_بjson(json!({
            "بحث": q,
            "الصفحة": page
        }))
    })
    .شغل(8080).await;
```

### معالجة JSON

```rust
use serde::{Deserialize, Serialize};

#[derive(Serialize, Deserialize)]
struct مستخدم {
    الاسم: String,
    البريد: String,
    العمر: u32,
}

خادم::جديد()
    .إرسال("مستخدمين", |طلب| {
        // تحليل JSON من الطلب
        let user: مستخدم = طلب.المحتوى_كjson().unwrap();
        
        // رد بـ JSON
        طلب.رد_بjson(json!({
            "تم": true,
            "المستخدم": user
        }))
    })
    .شغل(8080).await;
```

### الردود المتقدمة

```rust
خادم::جديد()
    .مسار("ردود", |طلب| {
        // نص عادي
        // رد::نص("مرحباً")
        
        // HTML
        // رد::html("<h1>مرحباً</h1>")
        
        // JSON
        // رد::json(json!({"نجاح": true}))
        
        // خطأ
        // رد::خطأ("غير موجود", 404)
        
        // إعادة توجيه
        // رد::توجه("/صفحة-جديدة")
        
        // مع رؤوس إضافية
        رد::json(json!({"بيانات": "قيمة"}))
            .أضف_رأس("X-Custom-Header", "قيمة")
            .مع_cors()
    })
    .شغل(8080).await;
```

### القوالب

```rust
use almarjaa_web::قالب;
use almarjaa_web::قوالب;

خادم::جديد()
    // قالب بسيط
    .مسار("قالب", |_| {
        قالب::جديد("
            <h1>{{العنوان}}</h1>
            <p>{{المحتوى}}</p>
        ")
        .متغير("العنوان", "مرحباً")
        .متغير("المحتوى", "هذا قالب بسيط")
        .إلى_رد()
    })
    // قالب خطأ
    .مسار("خطأ", |_| {
        قوالب::صفحة_خطأ(404, "الصفحة غير موجودة")
    })
    .شغل(8080).await;
```

### الجلسات

```rust
use almarjaa_web::مخزن_الجلسات;
use std::sync::Arc;

let sessions = Arc::new(مخزن_الجلسات::جديد());

خادم::جديد()
    .مسار("تسجيل-دخول", |طلب| {
        let mut session = مخزن_الجلسات::جديد().أنشئ();
        session.دخول("123", "أحمد");
        session.عيّن("دور", "مدير");
        
        رد::json(json!({
            "session_id": session.المعرف
        }))
    })
    .شغل(8080).await;
```

## 📖 الوثائق الكاملة

### الطلب (طلب)

| الدالة | الوصف |
|--------|-------|
| `الطريقة` | طريقة HTTP (جلب، إرسال، ...) |
| `المسار` | المسار المطلوب |
| `معامل(الاسم)` | الحصول على معامل من المسار |
| `استعلام(الاسم)` | الحصول على معامل استعلام |
| `رأس(الاسم)` | الحصول على رأس |
| `المحتوى_كنص()` | المحتوى كنص |
| `المحتوى_كjson()` | تحليل المحتوى كـ JSON |
| `كوكي(الاسم)` | الحصول على كوكي |
| `ip_العميل()` | عنوان IP للعميل |

### الرد (رد)

| الدالة | الوصف |
|--------|-------|
| `نص(المحتوى)` | رد نصي |
| `html(المحتوى)` | رد HTML |
| `json(البيانات)` | رد JSON |
| `نجاح(البيانات)` | رد نجاح |
| `خطأ(الرسالة, الكود)` | رد خطأ |
| `توجه(العنوان)` | إعادة توجيه |
| `أضف_رأس(الاسم, القيمة)` | إضافة رأس |
| `مع_cors()` | تفعيل CORS |

### الوسائط الجاهزة

```rust
// تسجيل الطلبات
وسیطة::تسجيل()

// التحقق من Bearer Token
وسیطة::bearer_token()

// التحقق من API Key
وسیطة::api_key("المفتاح_السري")

// حد الطلبات
وسیطة::حد_الطلبات(1000)

// JSON فقط
وسیطة::json_فقط()

// حد الحجم
وسیطة::حد_الحجم(1024 * 1024) // 1MB

// وضع الصيانة
وسیطة::صيانة("الموقع في صيانة")
```
